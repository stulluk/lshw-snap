# lshw-snap
Simple snap for lshw

# Building

Run ```snapcraft``` or ```snapcraft --use-lxd```

# Installing

Run

```
sudo snap install lshw_1.0.snap --devmode
```

# Running

```
stulluk@ubuntu:~$ sudo lshw
```
